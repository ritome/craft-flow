# 顧客管理データ（委託先マスタ）シート構成

## シート概要

- シート名（例）：顧客管理
- 用途：委託販売先の基本情報（名前・住所・振込先など）を管理する

## 列定義

| 列 | 項目名（Excelのヘッダー） | 意味・説明 | 例 |
|----|---------------------------|------------|----|
| A  | 委託先ID                 | 委託先を一意に識別するID | 1001 |
| B  | 委託先名                 | 委託先の名称 | 〇〇商店 |
| C  | 住所                     | 郵便番号＋住所 | 〒020-0000 岩手県盛岡市… |
| D  | 振込銀行名               | 入金に使う銀行名 | 〇〇銀行 |
| E  | 振込支店名               | 支店名 | 盛岡支店 |
| F  | 口座種別                 | 普通／当座など | 普通 |
| G  | 口座番号                 | 銀行口座番号 | 1234567 |
| H  | 口座名義                 | 口座名義人 | カ）モリオカセンター |
| I  | 手数料率                 | この委託先に適用する手数料率（％） | 20 |
| …  | …                        | 必要に応じて増やす | … |

※ 列や項目名は、実際のスプレッドシートに合わせて修正してください。
