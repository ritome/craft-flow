# 委託販売精算書 テンプレートレイアウト

## 概要

- テンプレートファイルパス：`resources/excel/settlement_template.xlsx`
- シート名（例）：精算書
- 用途：1 委託先・1 請求期間ごとの精算書を発行するときのひな形

## ヘッダー部（上部）

| 項目           | 内容                     | 設定するセル例 |
|----------------|--------------------------|----------------|
| タイトル       | 委託販売精算書          | A1             |
| 請求期間ラベル | 期間：YYYY/MM/DD〜…     | A3             |
| 委託先名       | 顧客管理データの委託先名 | B5             |
| 委託先住所     | 顧客管理データの住所     | B6             |
| 発行日         | 精算書発行日             | H3             |

※ 実際のレイアウトに合わせてセル位置を修正してください。

## 商品明細テーブル（中央）

- 明細の開始行：例）行 10 から
- 列定義：

| 列 | 項目     | 説明                     | セル例（1行目） |
|----|----------|--------------------------|------------------|
| A  | No       | 行番号                   | A10              |
| B  | 商品名   | 商品名                   | B10              |
| C  | 単価     | 単価                     | C10              |
| D  | 数量     | 数量                     | D10              |
| E  | 売上金額 | 単価×数量               | E10              |
| F  | 手数料率 | 手数料率（％）           | F10              |
| G  | 手数料額 | 売上金額×手数料率／100   | G10              |
| H  | 精算額   | 売上金額−手数料額 など   | H10              |

- 2行目以降（A11〜Hxx）に同じ形式で繰り返し

## 合計部（下部）

| 項目           | 内容                     | セル例 |
|----------------|--------------------------|--------|
| 売上金額合計   | 明細の売上金額の合計     | E20    |
| 手数料額合計   | 明細の手数料額の合計     | G20    |
| 精算額合計     | 明細の精算額の合計       | H20    |
| 振込金額       | 精算額合計と同じ or 調整 | H22    |

※ 行番号・セルはテンプレートの実レイアウトに合わせて修正してください。

## 振込先情報

| 項目       | 内容                     | セル例 |
|------------|--------------------------|--------|
| 銀行名     | 顧客管理データの銀行名   | B25    |
| 支店名     | 顧客管理データの支店名   | B26    |
| 口座種別   | 普通／当座               | B27    |
| 口座番号   | 顧客管理データの口座番号 | B28    |
| 口座名義   | 顧客管理データの口座名義 | B29    |

## 備考

- 実際の Excel テンプレートを見ながら、「どのセルにどの値を入れるか」をここにメモしておく。
- Laravel側では、この仕様に基づいて「セル座標 → DTO のプロパティ」のマッピングを行う。
