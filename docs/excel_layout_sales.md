# 委託販売売上データ シート構成

## シート概要

- シート名（例）：売上データ
- 用途：日々の販売実績を記録し、月次の委託精算書の元データとする

## 列定義

| 列 | 項目名（Excelのヘッダー） | 意味・説明 | 例 |
|----|---------------------------|------------|----|
| A  | 日付                     | 販売日 | 2025/10/01 |
| B  | 委託先ID                 | 顧客管理データの委託先ID | 1001 |
| C  | 委託先名                 | 委託先名称（IDと一致している想定） | 〇〇商店 |
| D  | 商品コード               | 商品を識別するコード | P001 |
| E  | 商品名                   | 商品の名称 | 手づくりクッキー |
| F  | 単価                     | 1個あたりの販売単価 | 300 |
| G  | 販売数量                 | 販売した数量 | 10 |
| H  | 売上金額                 | 単価×数量 | 3000 |
| I  | 手数料率                 | 手数料率（％）。空の場合は顧客マスタ側を使うなど | 20 |
| J  | 備考                     | メモ等 | イベント販売 など |
| …  | …                        | 必要に応じて増やす | … |

※ 実際の列名・順番・有無は、スプレッドシートに合わせて修正してください。

2025/10/01,1002,△△商店,P002,さきいか,500,5,2500,15,イベント販売
